# create-augur

Scaffold a new [Augur](https://augur.run) project — your second brain, on your machine.

## Usage

    npx create-augur@latest my-brain

## Requirements

- Node.js 20+
- Python 3.11+ (for MCP server and skills)
- pnpm (for dashboard)
- uv (for Python dependencies)
