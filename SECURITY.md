# Security Policy

Augur is currently in soft launch. Native macOS support is implemented, native Windows architecture is implemented, and Windows validation is still pending before we make a firmer public support claim.

## Supported Versions

At this stage, security fixes are tracked against the current `main` branch and the active soft-launch codebase.

## Reporting a Vulnerability

If you discover a vulnerability, please report it privately rather than opening a public issue.

Use one of the following:

1. The GitHub Security Advisory "Report a vulnerability" flow.
2. Email `security@augur-os.dev`.

We will acknowledge reports promptly and provide a remediation timeline when possible.

## Policy

- Please allow reasonable time to address the issue before public disclosure.
- Reporters who follow the policy will receive credit where appropriate.
